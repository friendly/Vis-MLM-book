Quarto maintains an LLM-friendly documentation index at https://quarto.org/llms.txt. Use this to locate authoritative Quarto documentation before answering questions about Quarto features, flags, or behavior — especially to avoid hallucinating options that don't exist (e.g., the `--config` flag that doesn't exist in `quarto render`).
